### 1. PrismML-Eng/Bonsai-demo 📈 +196 stars (总计 1,515)

- **简介**：Prism ML 推出的 Bonsai 系列语言模型的本地推理 Demo，支持 1-bit 和 Ternary（三值）量化的小型 LLM，可在 Mac（Metal）、Linux/Windows（CUDA/Vulkan）甚至 iPhone 上运行。最新发布 Bonsai 27B 视觉语言模型。
- **增长原因**：持续活跃开发中（7月13-15日连续更新），新版 Demo 支持 OpenWebUI agentic 体验、社区基准测试（DGX Spark 上 44 tokens/s）、Windows 支持改进。1-bit LLM 在端侧设备上运行的吸引力持续扩大。
- **核心特点**：极低比特量化（~1.125 bits/weight），27B 模型可装入现代 iPhone；支持视觉理解、工具调用、MCP 服务、256k+ 长上下文；`./setup.sh` 一行命令即可运行。
- **应用场景**：希望在本地设备上运行 LLM 的个人开发者、隐私敏感场景、端侧 AI 研究人员。
- **风险/局限**：模型仓库目前还是私有的（需 HF token）；1-bit 量化精度有损，非通用场景的最佳选择。

### 2. ibelick/ui-skills 📈 +178 stars (总计 4,264)

- **简介**：面向 Design Engineer 的 UI 技能 CLI 工具，通过 `npx ui-skills start` 让 AI 代理根据任务自动路由到正确的 UI 技能集（动画、布局、交互等）。
- **增长原因**：近期密集开发中（7月13-16日连续更新），新增 featured skills 集合、AGENTS.md 指导文档、SEO 优化，以及多个新技能卡片。TypeScript 技能类项目在 AI 编程社区持续热门。
- **核心特点**：`npx ui-skills` 零配置使用；将 UI 设计经验编码为可复用的 AI 技能指令；按类别筛选（motion, baseline-ui 等）。
- **应用场景**：使用 AI 编码工具（Claude Code、Copilot 等）的前端工程师/Design Engineer，希望让 AI 更懂 UI 设计规范。
- **风险/局限**：依赖 npm 生态和 AI 编码工具的支持；技能库的质量取决于维护者的持续更新。

### 3. ossu/computer-science 📈 +107 stars (总计 206,513)

- **简介**：Open Source Society University — 基于在线免费资源的自学计算机科学完整课程体系，从入门到高级再到毕业项目，覆盖 CS 本科核心课程。
- **增长原因**：持续稳定增长，作为 GitHub 上最著名的自学 CS 课程之一（20万+ stars），每天都有新生加入学习。近期有链接修复和资源更新。
- **核心特点**：完全免费、自定进度、社区支持；课程来自 Harvard、MIT、Princeton 等名校；设计为 2 年完成（每周 20 小时）。
- **应用场景**：希望系统学习 CS 但无法/不想上大学的自学者；转行程序员；需要 structured CS 基础的 AI/ML 从业者。
- **风险/局限**：需要极强的自律性；无官方学位；部分课程可能需要支付认证费用。

### 4. PostHog/posthog 📈 +77 stars (总计 35,836)

- **简介**：开源的自驱型产品分析平台，提供产品分析、会话回放、A/B 测试、Feature Flags、错误追踪、日志、AI 可观测性等全栈产品工具。支持通过 Slack/桌面/MCP 操控。
- **增长原因**：持续活跃的高频开发（最近24小时内有10+ commits），发布 agent-skills v0.495.0 版本、新增多个数据仓库导入源（Checkmarx、Instana、Gerrit 等）、增强 Agent Platform MCP 工具集成。
- **核心特点**：开源可自托管 vs 云服务双模式；自驱模式可自动从产品数据中发现错误并生成修复 PR；覆盖产品分析的完整链路：追踪→分析→实验→修复。
- **应用场景**：需要完整产品分析和 AI 可观测性的 SaaS 团队；希望用数据驱动产品决策的创业公司；自托管场景的数据隐私敏感企业。
- **风险/局限**：自托管部署有一定复杂度；功能丰富但小团队可能需要较长的上手时间。

### 5. lobehub/lobehub 📈 +71 stars (总计 80,156)

- **简介**：LobeHub — 你的首席 Agent 运维官，将 AI 代理组成 7×24 小时运行的团队，支持招聘、调度和报告。支持 OpenAI、Claude、DeepSeek、Gemini 等多模型。

- **核心特点**：多模型/多供应商统一接入；Agent 编排和调度框架；支持知识库、MCP、技能系统；开源可自托管。
- **应用场景**：需要统一管理多个 AI Agent 的团队；构建多 Agent 协作系统的开发者；想自建 AI 工作流的个人/企业。
- **风险/局限**：功能迭代极快，可能存在稳定性波动；需一定的技术能力来部署和维护。

- **简介**：Apache 孵化项目，致力于制定跨 AI、分析和 BI 平台的语义模型交换标准。提供供应商中立的 JSON/YAML 规范，确保元数据在不同工具间一致可用。
