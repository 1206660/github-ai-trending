## 今日 Top Stars 增长榜

### 1. JuliusBrussee / caveman 📈 +926 stars (总计 80,953)

- **简介**：🪨 "why use many token when few token do trick" — 一个让 AI 编码助手（Claude Code、Codex、Cursor 等）用"原始人语"输出以减少 Token 消耗的 Skill，平均压缩输出 65%。
- **增长原因**：病毒式的 meme 吸引力（"多 token 不如少 token"）+ 切实的价值（省钱+加速）。登顶 Hacker News #1，并获 OpenAI、微软、Google 等大厂工程师关注。v1.9.0 发布修复安全漏洞，持续活跃开发。
- **核心特点**：4 级压缩模式（lite/full/ultra/wenyan）、支持 30+ 个 AI 编码 Agent、提供 commit/review/stats/compress 等子命令、一键安装、MIT 开源。
- **应用场景**：使用 AI 编码助手的开发者，希望减少 Token 消耗、加速响应、节省 API 费用。
- **风险/局限**：输出可读性下降（尤其 ultra 模式），需 Node ≥18，仅 3 个月历史的早期项目。

### 2. openai / codex-plugin-cc 📈 +352 stars (总计 22,639)

- **简介**：OpenAI 官方出品的 Claude Code 插件，让你在 Claude Code 中直接调用 Codex 进行代码审查或任务委派。
- **增长原因**：v1.0.5 发布新增 `/codex:transfer` 命令（可将 Claude Code 会话无缝移交至 Codex）。OpenAI 为 Anthropic 产品做插件这件事本身就有新闻价值。
- **核心特点**：8 个斜杠命令（review、adversarial-review、rescue、transfer 等）、Review Gate 自动审查门控、支持后台异步任务。
- **应用场景**：Claude Code 用户希望获得 Codex 的高质量代码审查能力，或在两者之间灵活切换。
- **风险/局限**：同时依赖 ChatGPT 订阅和 Claude Code，消耗双方使用配额，云端依赖。

### 3. langflow-ai / langflow 📈 +117 stars (总计 150,737)

- **简介**：可视化拖拽式 AI 代理工作流构建工具，支持部署为 API 或 MCP Server。150K+ Stars 的成熟项目。
- **增长原因**：v1.10 发布带来重大更新（AI Assistant 流构建、Memory Bases、Knowledge Bases 重构），Desktop 版上线降低门槛。MCP 标准的大流行让 Langflow 的 MCP Server 部署能力成为热点。
- **核心特点**：可视化 Flow Builder、多 Agent 编排、一键部署 API/MCP、Python 组件可自定义、支持 LangSmith/Grafana 可观测性。
- **应用场景**：希望以可视化方式构建 AI Agent 工作流并在生产环境部署的开发者。
- **风险/局限**：复杂流程的可视化编辑上限、强依赖 LangChain 生态、云版本可能领先开源版。

### 4. ChromeDevTools / chrome-devtools-mcp 📈 +104 stars (总计 45,086)

- **简介**：Google ChromeDevTools 团队官方出品的 MCP Server，让 AI 编码 Agent 直接控制 Chrome 浏览器进行调试、自动化、性能分析。
- **增长原因**：v1.4.0 发布 + 持续活跃开发（今天就有 4 个 commit）。作为浏览器自动化的 MCP 标准，随 AI Agent 生态爆发而快速增长。
- **核心特点**：44+ 工具涵盖输入自动化、导航、性能追踪、网络分析、内存堆快照、Lighthouse 审计、扩展管理。支持 slim 轻量模式。
- **应用场景**：AI Agent 需要控制浏览器进行测试、调试、性能分析或网页自动化。
- **风险/局限**：仅官方支持 Chrome，安全风险（Agent 可完全控制浏览器），部分功能需特定 Chrome 版本。

### 5. agentskills / agentskills 📈 +86 stars (总计 21,618)

- **简介**：Agent Skills 开放格式规范——用 `SKILL.md` + 脚本/参考文件的可移植文件夹扩展 AI Agent 的能力。源自 Anthropic。

- **应用场景**：开发者编写可复用、可分享的 Agent 技能；Agent 平台采用统一技能标准。
