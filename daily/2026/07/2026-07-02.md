### 1. allenai/olmocr 📈 +334 stars (总计 18,272)

- **简介**：AI2（Allen Institute for AI）开源的 PDF 线性化工具包，使用视觉语言模型将 PDF 和图像文档转换为干净的 Markdown/纯文本，专为 LLM 训练数据管道设计。
- **增长原因**：非近期发布驱动，而是持续稳定出现在 GitHub Trending 页面带来自然增长。OCR 社区近期工具对比热议（Chandra OCR 等新竞争者入局），以及 Google News 上关于 olmOCR 的技术报道带来了额外流量曝光。

- **应用场景**：大规模 PDF 转 LLM 训练数据集、学术论文数字化、企业文档解析、OCR 基准对比研究。
- **风险/局限**：需 GPU（最低 12GB VRAM）；近 3 个月无代码更新，处于维护期；在 olmOCR-Bench 上得分 82.4，略低于新秀 Chandra OCR 的 83.1。

### 2. TencentCloud/CubeSandbox 📈 +79 stars (总计 6,799)

- **简介**：腾讯云开源的 AI Agent 沙箱服务，用 Rust 编写，基于 KVM/RustVMM 微虚拟机提供硬件级隔离环境，是 E2B 的 Drop-in 开源替代品。
- **增长原因**：项目迭代速度极快（不到 3 个月发布 4 个 major 版本）；v0.4.0 新增 CubeEgress 安全代理和凭据保险库；被 CNCF Landscape 收录；AI Agent 社区对安全自托管沙箱需求旺盛，E2B 替代品话题持续升温。
- **核心特点**：冷启动 < 60ms，单实例内存开销 < 5MB；P99 < 150ms 在 50 并发条件下；支持快照/克隆/回滚（Copy-on-Write）；eBPF 虚拟交换机实现网络隔离；完全兼容 E2B API，只需替换 URL 即可迁移。
- **应用场景**：AI Agent 执行不可信代码的安全沙箱、LLM 生成代码的隔离运行环境、企业级多租户代码执行平台。
