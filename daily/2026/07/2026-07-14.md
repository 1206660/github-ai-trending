## 今日 Top Stars 增长榜

### 1. OpenCut-app/OpenCut 📈 +1,229 stars (总计 66,372)

- **简介**：开源免费的视频编辑器，号称 CapCut 的替代品。支持 Web、桌面和移动端，使用 Rust 核心实现跨平台。
- **增长原因**：项目正在进行**从零重写（rewrite）**，准备推出一代全新架构。最新提交（7月10日）显示正在构建桌面版应用（scaffold desktop app），架构升级包括 Editor API、第三方插件系统、MCP Server 等重磅特性。这次重写引发了社区的广泛关注和期待，大量用户 star 标记等待正式发布。
- **核心特点**：Rust 核心 + 跨平台（Web/桌面/移动端）；正在开发插件优先架构和 MCP 服务器（AI Agent 集成）；支持头戴模式（自动化、批量渲染）；编辑器内脚本面板。
- **应用场景**：需要免费开源视频编辑的个人创作者、自媒体；寻找 CapCut 替代品的用户；对 AI 辅助视频编辑（MCP 集成）感兴趣的开发者。
- **风险/局限**：目前仍在重写中，稳定版本需等待；经典版保留了旧分支（opencut-classic）；当前暂不接受外部贡献，架构尚在设计中。

### 2. Graphify-Labs/graphify 📈 +1,095 stars (总计 84,727)

- **简介**：AI 编程助手的知识图谱技能。可将任意文件夹中的代码、SQL 模式、脚本、文档、论文、图片或视频转换为可查询的知识图谱。支持 Claude Code、Codex、Cursor、Gemini CLI 等 15+ 平台。
- **增长原因**：近期连续发布多个补丁版本（v0.9.11~v0.9.15），修复了包括存储型 XSS 安全漏洞、文件遍历 bug、PostgreSQL 外键检测等关键问题。频繁的修复版本迭代 + 持续的社区 PR 贡献，加上 Y Combinator S26 的背书，使其持续获得高关注度。
- **核心特点**：基于 tree-sitter AST 的代码解析，完全本地运行，不发送代码到外部；真正的知识图谱（非向量索引），支持边的关系追踪（EXTRACTED/INFERRED 标记）；支持图型界面 graph.html 交互浏览。
- **应用场景**：大型代码库的导航和理解；AI 编程助手的上下文增强；跨项目代码分析；帮助新成员快速上手复杂项目。
- **风险/局限**：84K stars 已处于成熟期，增速可能放缓；依赖外部 AI 模型的 LLM 调用（文档/图片/视频分析）；508 个 open issues 显示维护压力较大。

### 3. github/spec-kit 📈 +543 stars (总计 120,593)

- **简介**：GitHub 官方推出的 Spec-Driven Development（规范驱动开发）工具包，让规范（Spec）成为可执行的开发指南，直接生成工作实现而不仅仅是指引。
- **增长原因**：近期高频发布 v0.12.10~v0.12.14 版本（7月10日至13日间连续推送），社区贡献活跃（Test-First Governance 预设、Spec Kit Memory 扩展等）。作为一个拥有 120K+ stars 的巨型项目，其稳定的日常增速本身就来自 GitHub Trending 的自增强循环。
- **核心特点**：规范即代码——Spec 可执行、可直接生成实现；支持 30+ AI 编程代理集成（Claude Code、Copilot、Codex 等）；提供 `/speckit.*` 命令集（constitution→specify→plan→tasks→implement 完整流程）。
- **应用场景**：需要结构化开发的 AI 编程团队；从"氛围编程（vibe coding）"转向规范驱动开发的团队；企业级项目的标准化开发流程。
- **风险/局限**：120K stars 意味着项目的 Trending 热度主要来自已有 user base 的自增强循环，而非突破性新功能；需要 ux/pipx 等工具链支持，有一定上手门槛。

- **核心特点**：自托管（用户拥有数据），支持实时语音对话；可接入 Minecraft/Factorio 游戏控制；支持 Live2D/VRM 虚拟形象；丰富的多语言支持（中英日韩俄法等）。
- **应用场景**：AI VTuber/虚拟主播爱好者；Neurosama 风格 AI 伴侣体验探索者；自托管 AI 语音交互实验。
