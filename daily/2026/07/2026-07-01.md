- **简介**：用 Rust 编写的 AI Agent 终端多路复用器——可以理解为"为 AI 编码 Agent 重造的 tmux"——约 10MB 单二进制文件，跨平台终端内运行。
- **增长原因**：7 小时前发布最新预览版（新增 Shell 补全、隐藏侧栏、会话快照 API 等 11 项功能）；上周 v0.7.1 稳定版带来了成熟度跃升。原生支持 15+ 主流编码 Agent（Claude Code、Codex、Copilot CLI、Hermes Agent 等）。
