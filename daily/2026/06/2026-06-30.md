## 今日新项目分析

### 1. 0xNyk/council-of-high-intelligence 📈 +331 stars（总计 1.9k）

- **简介**：一个多 LLM 决策委员会工具，让你召集 18 位"AI 智者"（亚里士多德、费曼、卡尼曼、托瓦兹等）就你的难题进行结构化多轮辩论，最终给出加权投票结论。
- **增长原因**：概念极具病毒传播力——"让 AI 名人替你决策"在社交媒体上引发大量讨论。项目持续迭代，最近 3 天前还有 tie-breaking 投票算法优化提交。
- **核心特点**：跨平台多 LLM 支持（Claude、Gemini、OpenAI、Ollama 自动路由），18 个独特人格角色各有独立 prompt 和思维模式，结构化多轮辩论而非简单聚合，支持 `/council` 一键命令。
- **应用场景**：个人决策辅助（职业选择、投资决策）、创意 brainstorming、作为 multi-agent 系统的 Go-to demo。
- **风险/局限**：决策质量受底层 LLM 能力限制；多轮辩论耗时较长且消耗较多 token；人格角色本质上仍是 prompt engineering 产物。

### 2. veracrypt/VeraCrypt 📈 +186 stars（总计 10.5k）

- **简介**：基于 TrueCrypt 的磁盘加密工具，支持全盘加密、隐藏卷、操作系统预启动认证，是目前最流行的开源全盘加密方案之一。
- **增长原因**：2026年6月9日发布 **VeraCrypt 1.26.29**，修复了两个安全漏洞（CVE-2026-53762、CVE-2026-54073），新增 **Argon2id KDF** 支持，修复 Windows 11 25H2 休眠崩溃问题，带动了新一轮关注。
- **核心特点**：支持 AES/Twofish/Serpent/Camellia 等多种加密算法、系统分区全盘加密、隐藏卷（plausible deniability）、跨平台（Windows/macOS/Linux）、支持 PIM（Personal Iterations Multiplier）。
- **应用场景**：个人隐私保护、笔记本全盘加密、企业合规加密需求、U 盘加密便携。
- **风险/局限**：代码库较老（基于 2014 年的 TrueCrypt），架构现代化进程缓慢；Windows 驱动偶有 BSOD（1.26.29 修复了一部分）；社区规模相对较小。

### 3. logto-io/logto 📈 +158 stars（总计 12.7k）

- **简介**：面向 SaaS 和 AI 应用的身份认证与授权基础设施，基于 OIDC 和 OAuth 2.1，提供多租户、SSO、RBAC 等功能。
- **增长原因**：5月底发布 **v1.40.0/v1.40.1**，新增审计日志时间范围筛选器、大租户性能优化、新邮件/SMS 连接器（MailJunky、SMSBao、阿里云短信），以及数据库直连管理员签名密钥等 OSS 部署增强功能，吸引更多自托管用户。
- **核心特点**：开源替代 Auth0 的最佳选择之一，TypeScript 全栈开发、完整的 OIDC/OAuth 2.1 支持、内置 MFA/TOTP/SAML/SSO、可自托管、Cloud 版本一键部署。
- **应用场景**：SaaS 产品快速集成认证系统、AI 应用用户管理、企业多租户身份方案、需要自托管控数据的场景。
- **风险/局限**：生态系统相比 Keycloak 仍较年轻；部分高级功能（如企业 SSO）仅 Cloud 版提供；社区插件/连接器数量有限。

### 4. Unclecheng-li/VulnClaw 📈 +129 stars（总计 1.1k）

- **简介**：基于 AI Agent + MCP 工具链 + 渗透技能编排的自动化安全测试框架。输入自然语言即可自动完成「信息收集 → 漏洞发现 → 漏洞利用 → 报告生成」全流程。
- **增长原因**：AI 驱动的自动化渗透测试是 2026 年持续火爆的话题。VulnClaw 将 MCP（Model Context Protocol）工具链与安全渗透技能编排结合，实现了从"自然语言指令"到"完整安全报告"的端到端自动化，概念吸引力极强。
- **核心特点**：自然语言交互（"帮我扫描这个网站"）、自动调用渗透测试工具链、MCP 协议实现工具编排、自动生成渗透测试报告、支持 CTF 场景。
- **应用场景**：安全研究员快速 Recon & 渗透、CTF 竞赛加速、企业安全团队的自动化辅助、AI Agent 安全能力 demo。
- **风险/局限**：自动化渗透可能产生误报；依赖底层 LLM 对安全场景的理解能力；可能在未经授权的系统上使用带来法律风险；仍处于早期阶段（1.1k stars）。
