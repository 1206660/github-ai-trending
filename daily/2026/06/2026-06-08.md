### 1. RyanCodrai/turbovec 📈 +1,554 stars (总计 7,190)
- **简介**：基于 Google TurboQuant 算法的高性能向量检索引擎，Rust 实现 + Python 绑定，10M 向量仅占 4GB 内存。
- **增长原因**：今日暴涨可能与 v0.7.0 Python + v0.8.0 Rust 版本联合发布（5月30日）带来的社区关注延续；TurboQuant 论文刚发布 arXiv，正处于技术社区热议期。
- **核心特点**：数据遗忘型量化器（data-oblivious quantizer），无需训练、无 codebook；手写 NEON/AVX-512BW 内核，比 FAISS 快 12-20%；搜索时支持 ID 白名单过滤；纯本地运行，支持 air-gapped RAG。
- **应用场景**：隐私敏感或内存受限的 RAG 场景（医疗、法律、金融）；边缘设备向量检索；需快速部署的开源向量搜索基础设施。
- **风险/局限**：生态比 FAISS 窄（刚起步）；生产验证案例有限；Rust + Python 混合栈对纯 Python 团队有门槛。
