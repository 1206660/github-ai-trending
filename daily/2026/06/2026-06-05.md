### 🆕 1. openclaw/openclaw-windows-node 📈 +411 stars (总计 1,326)
- **简介**：OpenClaw 的 Windows 节点版，让 AI Agent 能在 Windows 上运行，实现跨平台"数字分身"
- **增长原因**：OpenClaw 是近期火热的本地 AI Agent 部署方案，继 DeepSeek/Manus 之后受到全网刷屏，Windows 版刚发布吸引大量 Windows 用户关注
- **核心特点**：一键安装脚本、支持 PowerShell、管理员权限自动处理、多 AI 提供商后端
- **应用场景**：想在 Windows 本地部署 AI Agent 的开发者；需要跨平台数字分身能力的用户
- **风险/局限**：项目较新（1,326 stars），成熟度待验证；依赖 Node.js 环境；Windows 专有版本可能存在兼容性问题

### 🆕 2. aquasecurity/trivy 📈 +255 stars (总计 35,668)
- **简介**：开源容器安全扫描工具，可发现漏洞、错误配置、 secrets 和 SBOM，支持容器、Kubernetes、云等多种场景
- **增长原因**：近期容器安全事件频发，加上 DevSecOps 理念普及，安全扫描类工具需求增长；该工具被纳入 CNCF landscape，可信度高
- **核心特点**：开箱即用、无需配置数据库、支持多语言/多平台、SBOM 生成、CI/CD 集成、与主流云厂商兼容
- **应用场景**：DevOps/SecOps 团队；CI/CD 流水线安全扫描；容器镜像合规检查；Kubernetes 安全审计
- **风险/局限**：偏向发现而非修复；大量误报需要人工筛选；对私有漏洞库支持有限

### 🆕 3. lfnovo/open-notebook 📈 +212 stars (总计 25,033)
- **简介**：开源 NotebookLM 替代品，支持 16+ AI 提供商，可管理研究工作流、生成 AI 笔记、播客和总结
- **增长原因**：Google NotebookLM 功能强大但封闭，开源替代方案需求强烈；支持多 AI 后端灵活性高
- **核心特点**：多 AI 模型支持（OpenAI/Anthropic/Ollama 等）、多笔记本管理、播客自动生成、搜索引擎集成、细粒度上下文管理、支持多种文件格式
- **应用场景**：研究人员、学生、内容创作者；需要多源 AI 辅助笔记的个人知识管理；NotebookLM 粉丝寻找开源替代
- **风险/局限**：自托管需配置 API 密钥，成本由用户承担；部分高级功能依赖付费模型；隐私数据需自行保障
