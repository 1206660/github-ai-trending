### 1. microsoft / markitdown 📈 +3,618 stars (总计 ~141K)
- **简介**：Microsoft 出品的 Python 工具，将文件和 Office 文档转换为 Markdown 格式
- **增长原因**：文档转 Markdown 是刚需场景，且来自微软官方背书，质量有保证；近期 AI 应用爆发，大量项目需要处理文档数据
- **核心特点**：支持多种格式（PDF/Word/Excel/PPT 等），转换质量高，保持格式和目录结构
- **应用场景**：AI 应用开发、RAG 系统构建、知识管理、内容处理流水线
- **风险/局限**：依赖 Microsoft 生态，复杂格式转换可能有信息损耗

- **增长原因**：AI 编码工具军备竞赛，ECC 提供 Skills/Instincts/Memory/Security 等优化层

### 4. chopratejas / headroom 📈 +1,265 stars (总计 ~6.4K)
- **简介**：AI Agent 的上下文压缩层，Token 减少 60-95%，效果不变
- **增长原因**：大模型 Token 成本高，压缩需求强烈；MCP 协议加持让它能作为 MCP Server 无缝接入各种 Agent
- **核心特点**：
  - 6 种压缩算法（SmartCrusher/CodeCompressor/Kompress-base 等）
  - 三种形态：Library、Proxy、MCP Server
  - 支持跨 Agent 共享记忆（Claude/Codex/Gemini）
  - `headroom learn` 从失败会话学习，自动写 CLAUDE.md
- **应用场景**：降低 LLM 调用成本、提升 AI Agent 效率、RAG 场景、上下文窗口管理
- **风险/局限**：压缩算法可能有信息丢失风险；本地部署数据安全但配置复杂度高
