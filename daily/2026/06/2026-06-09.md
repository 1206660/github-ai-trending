## 🆕 新增分析项目（今日首次分析）

### 1. roboflow/supervision 📈 +1,288 stars (总计 42,349)
- **简介**：Roboflow 出品的开源计算机视觉工具库，提供可视化、跟踪、计数、格式转换等常用工具，与 YOLO、Detectron2、Transformers 等模型无缝对接
- **增长原因**：计算机视觉开发者苦于每次项目重复写绘图/标注/计数逻辑；Supervision 提供统一 API 解决格式转换痛点；Roboflow 品牌背书+活跃社区
- **核心特点**：
  - MIT 协议，完全开源
  - 支持 YOLO、Detectron2、Transformers、Inference 等主流模型
  - 丰富的标注器（bounding box、mask、line 等）
  - 数据集加载器，支持多种格式
  - 安装简单：`pip install supervision`
- **应用场景**：
  - 视频分析/物体追踪
  - 区域计数/热力图
  - 数据集处理流水线
  - AI 视觉应用快速原型
- **风险/局限**：
  - 强依赖 Roboflow 生态，部分功能需要 API key
  - 文档以英文为主，中文社区资源较少
  - 主要面向有 ML 基础的开发者
